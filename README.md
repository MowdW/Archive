# 档案库插件 (Archive Folders)

一个美观的物理文件夹风格文件管理器，为 Obsidian 提供直观高效的文件管理体验。

## 功能特性

- 🎨 **物理文件夹风格** - 模拟真实文件夹的视觉效果和交互体验
- 📁 **3D 打开动画** - 文件夹打开时的流畅 3D 动画效果
- 🗂️ **文件夹导航** - 使用键盘快捷键在文件夹间导航（Enter 进入，Backspace 返回，Escape 根目录）
- 🎯 **拖拽排序** - 自由拖拽文件和文件夹进行排序和组织
- ✅ **多选操作** - 支持 Shift+点击和框选进行批量操作
- 📐 **自动布局** - 提供网格、圆形、树状图三种自动布局方式
- 🎨 **自动配色** - 一键为选中项目自动分配协调的颜色
- 📏 **对齐功能** - 支持左对齐、顶部对齐、水平等距、垂直等距
- 🧲 **网格吸附** - 启用网格吸附功能，使元素对齐更加精准
- 🔍 **面包屑导航** - 显示当前路径，支持点击导航到任意层级
- ⌨️ **键盘快捷键** - 丰富的键盘快捷键支持

## 安装方法
### 手动安装
1. 下载最新版本的插件文件
2. 将插件文件解压到 Obsidian 的插件目录中
3. 重启 Obsidian
4. 在设置中启用插件

## 使用指南

### 基本操作
- **创建新档案**：点击 Obsidian 工具栏中的 "新建档案" 按钮（inbox 图标）
- **打开档案**：双击 .arc 文件或在 Obsidian 中打开
- **添加文件**：点击界面中的 "新建文件夹" 或 "新建文档" 按钮
- **添加媒体**：拖拽图片或其他媒体文件到界面中

### 键盘快捷键
- **Enter**：进入选中的文件夹
- **Backspace/Delete**：返回上一级文件夹
- **Escape**：返回根目录
- **Shift+点击**：多选文件和文件夹
- **拖拽**：移动文件和文件夹位置

### 布局操作
- **网格布局**：将选中项目排列成网格
- **圆形布局**：将选中项目排列成圆形
- **树状图布局**：将选中项目排列成树状结构
- **对齐**：使用工具栏中的对齐按钮进行精确对齐
- **自动配色**：一键为选中项目分配协调的颜色

### 导航操作
- **面包屑导航**：点击面包屑中的任意层级快速导航
- **路径栈**：使用键盘快捷键在文件夹间导航

## 设置选项

### 新建文件保存位置
- 在插件设置中可以设置通过插件创建的新文件的保存位置
- 留空则使用默认位置（"档案" 文件夹）

## 示例

### 创建新档案
1. 点击 Obsidian 工具栏中的 "新建档案" 按钮
2. 系统会在 "档案" 文件夹中创建一个新的 .arc 文件
3. 双击打开新创建的档案文件

### 组织文件
1. 点击 "新建文件夹" 按钮创建文件夹
2. 点击 "新建文档" 按钮创建文档
3. 拖拽文件到文件夹中进行组织
4. 使用自动布局功能快速整理文件

### 导航文件夹
1. 选择一个文件夹
2. 按 Enter 键进入文件夹
3. 按 Backspace 键返回上一级
4. 按 Escape 键返回根目录

## 常见问题

### Q: 如何创建新的档案文件？
A: 点击 Obsidian 工具栏中的 "新建档案" 按钮（inbox 图标），系统会在 "档案" 文件夹中创建一个新的 .arc 文件。

### Q: 如何在文件夹间导航？
A: 选择文件夹后按 Enter 键进入，按 Backspace 键返回上一级，按 Escape 键返回根目录。

### Q: 如何使用自动布局功能？
A: 选中多个项目后，点击工具栏中的布局按钮（网格、圆形、树状图）即可应用相应的布局。

### Q: 如何调整新建文件的保存位置？
A: 在插件设置中设置 "新建文件保存位置" 选项。
